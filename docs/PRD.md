# 김추페 (KimchuPa) - Product Requirements Document

## 1. 개요

### 1.1 제품 비전
김추페는 전 세계 김치 애호가를 위한 **김치 정보 & 커뮤니티 플랫폼**입니다. AI 기반 개인화 추천, 김치 위키, 활발한 커뮤니티를 통해 김치 문화의 글로벌화를 선도하며, 제휴 링크를 통해 사용자가 원하는 김치를 쉽게 구매할 수 있도록 연결합니다.

### 1.2 제품명
- 한글: 김추페 (김치 추천 페이지의 약자)
- 영문: KimchuPa
- 슬로건: "Discover Your Perfect Kimchi"

### 1.3 핵심 가치
- **정보**: 김치에 대한 모든 것을 알려주는 위키
- **추천**: AI 기반 개인 맞춤 김치 추천
- **커뮤니티**: 김치 애호가들의 소통 공간
- **연결**: 제휴 링크를 통한 구매 연결 (직접 판매 X)

### 1.4 타겟 사용자
| 구분 | 설명 | 주요 니즈 |
|------|------|----------|
| 김치 입문자 | 김치에 관심을 갖기 시작한 사람들 | 김치 종류 학습, 입문 가이드, 쉬운 추천 |
| 김치 애호가 | 김치를 즐기고 더 알고 싶은 사람들 | 다양한 김치 정보, 구매처 추천, 커뮤니티 |
| 요리 마니아 | 김치 담그기/요리에 관심 있는 사람들 | 레시피, 노하우, 발효 팁, 커뮤니티 인정 |
| 해외 거주자 | 해외에서 김치를 찾는 한인/외국인 | 현지 구매처 정보, 다국어 지원 |
| 김치 전문가 | 김치 명인, 연구자, 생산자 | 지식 공유, 브랜드 홍보, 커뮤니티 영향력 |

## 2. 핵심 기능

### 2.1 AI 김치 추천 시스템
- **개인화 추천**: 취향 분석 기반 맞춤 김치 추천
- **퀴즈 기반 추천**: 매운맛, 발효도, 용도 등 질문을 통한 추천
- **활동 기반 추천**: 위키 열람, 커뮤니티 활동 데이터 기반 추천
- **시즌 추천**: 계절, 명절별 추천 김치

### 2.2 김치 위키 (KimchiPedia)
김치에 대한 모든 정보를 담은 백과사전

#### 2.2.1 김치 종류 DB
- **기본 정보**: 이름, 지역, 역사, 유래
- **특성**: 매운맛 레벨, 발효 정도, 주재료, 부재료
- **영양 정보**: 칼로리, 영양소, 건강 효능
- **보관법**: 적정 온도, 보관 기간, 숙성 팁
- **페어링**: 어울리는 음식, 요리 활용법

#### 2.2.2 김치 담그기 가이드
- **단계별 레시피**: 초보자/중급자/고급자 레벨별
- **재료 가이드**: 재료 선택법, 대체 재료
- **도구 가이드**: 필요한 도구, 추천 제품
- **트러블슈팅**: 실패 원인 및 해결법

#### 2.2.3 김치 문화
- **역사**: 김치의 기원과 발전
- **지역별 특색**: 지역별 김치 특징
- **세계 속 김치**: 글로벌 김치 문화
- **김치 명인**: 유명 김치 장인 소개

#### 2.2.4 사용자 기여
- **위키 편집 제안**: 정보 추가/수정 제안
- **편집자 등급**: 기여도에 따른 편집 권한
- **출처 관리**: 신뢰할 수 있는 정보 관리

### 2.3 제휴 링크 시스템 (Affiliate)
직접 판매 없이 외부 쇼핑몰로 연결하여 수수료 수취

#### 2.3.1 상품 연결
- **제휴 쇼핑몰**: 쿠팡, 네이버, 아마존, iHerb 등
- **상품 DB**: 김치 상품 정보 수집/관리
- **가격 비교**: 여러 쇼핑몰 가격 비교
- **재고 상태**: 품절 여부 표시

#### 2.3.2 추천 상품
- **AI 매칭**: 사용자 취향에 맞는 상품 추천
- **위키 연동**: 김치 종류별 구매 가능 상품 연결
- **리뷰 연동**: 사용자 리뷰 기반 추천

#### 2.3.3 링크 추적
- **클릭 추적**: 어떤 상품이 인기 있는지 분석
- **전환 추적**: 실제 구매로 이어진 비율
- **수수료 정산**: 제휴사별 수수료 관리

### 2.4 커뮤니티 & 레벨 시스템

#### 2.4.1 게시판
- **레시피 게시판**: 김치 담그기, 김치 요리 레시피
- **자유게시판**: 일상, 김치 이야기
- **Q&A**: 김치 관련 질문/답변
- **리뷰 게시판**: 김치 상품 구매 후기
- **김치 일기**: 발효 과정 기록/공유
- **지역별 게시판**: 지역 커뮤니티

#### 2.4.2 레벨 시스템
| 레벨 | 이름 | 필요 경험치 | 혜택 |
|------|------|------------|------|
| 1 | 김치 새싹 | 0 | 기본 기능 |
| 2 | 김치 입문자 | 100 | 댓글 작성 |
| 3 | 김치 애호가 | 500 | 게시글 작성, 리뷰 작성 |
| 4 | 김치 마니아 | 2,000 | 레시피 등록, 위키 편집 제안 |
| 5 | 김치 달인 | 5,000 | 위키 직접 편집, 뱃지 획득 |
| 6 | 김치 장인 | 15,000 | 전문가 마크, 칼럼 작성 |
| 7 | 김치 명인 | 50,000 | 명예의 전당, 특별 뱃지 |

#### 2.4.3 경험치 획득
| 활동 | 경험치 |
|------|--------|
| 출석 체크 | +5 |
| 게시글 작성 | +20 |
| 댓글 작성 | +5 |
| 좋아요 받음 | +2 |
| 레시피 등록 | +50 |
| 위키 편집 승인 | +30 |
| 김치 일기 작성 | +15 |
| 베스트 게시글 선정 | +100 |
| 퀴즈 완료 | +10 |
| 제휴 링크 클릭 | +1 |

#### 2.4.4 뱃지 시스템
- **활동 뱃지**: 연속 출석, 다작 작성자 등
- **전문 뱃지**: 레시피 전문가, 리뷰어 등
- **이벤트 뱃지**: 챌린지 참여, 대회 수상 등
- **특별 뱃지**: 초기 멤버, 기여자 등

#### 2.4.5 소셜 기능
- **팔로우/팔로워**: 관심 있는 사용자 팔로우
- **좋아요/북마크**: 게시글 저장
- **공유**: SNS 공유
- **DM**: 다이렉트 메시지 (레벨 4+)

#### 2.4.6 이벤트
- **김치 담그기 챌린지**: 월간/계절 챌린지
- **레시피 대회**: 창작 레시피 경연
- **김치 일기 챌린지**: 발효 과정 기록 챌린지
- **퀴즈 이벤트**: 김치 지식 퀴즈

### 2.5 교육 콘텐츠
- **비디오 튜토리얼**: 김치 담그기 강좌
- **초보자 가이드**: 김치 입문 코스
- **김치 투어 정보**: 한국 김치 명소, 체험 프로그램

## 3. 사용자 시나리오

### 3.1 김치 입문자 (해외 거주 외국인)
1. 웹사이트 접속 (영어 자동 감지)
2. "Find My Kimchi" 퀴즈 시작
3. 매운맛 레벨, 선호 재료, 용도 선택
4. AI가 입문자용 김치 3종 추천
5. 김치 위키에서 추천 김치 상세 정보 확인
6. "어디서 살 수 있나요?" 클릭 → 제휴 링크로 아마존 이동
7. 구매 후 커뮤니티에 리뷰 작성
8. 경험치 획득, 레벨업

### 3.2 김치 마니아 (국내 사용자)
1. 커뮤니티에서 레시피 탐색
2. 묵은지 담그기 챌린지 참여
3. 발효 과정 사진 업로드 (김치 일기)
4. 다른 사용자와 팁 교환, 좋아요 획득
5. 레벨업하여 "김치 달인" 달성
6. 위키 편집 권한 획득, 정보 기여
7. 베스트 레시피 선정, 특별 뱃지 획득

### 3.3 정보 탐색자
1. 검색엔진에서 "총각김치 담그는 법" 검색
2. 김추페 위키 페이지 도착
3. 상세한 레시피와 팁 확인
4. 관련 상품 제휴 링크 클릭
5. 회원가입 없이도 정보 열람 가능
6. 관심 생겨서 회원가입, 커뮤니티 참여

## 4. 비기능 요구사항

### 4.1 성능
- 페이지 로딩: 3초 이내 (First Contentful Paint)
- API 응답: 500ms 이내
- 동시 접속자: 10,000명 이상 지원
- 이미지 최적화: WebP, lazy loading
- SEO 최적화: 위키 페이지 검색엔진 노출

### 4.2 보안
- HTTPS 전 구간 암호화
- 개인정보: GDPR, 개인정보보호법 준수
- 스팸 방지: 레벨 기반 활동 제한, CAPTCHA
- XSS/CSRF 방어

### 4.3 국제화 (i18n)
- 지원 언어: 한국어, 영어, 일본어, 중국어(간체/번체), 스페인어
- 위키 다국어: 언어별 위키 페이지
- 제휴 링크: 국가별 쇼핑몰 연결

### 4.4 접근성
- WCAG 2.1 AA 기준 준수
- 스크린 리더 호환
- 키보드 네비게이션 지원

### 4.5 확장성
- 마이크로서비스 아키텍처
- 수평 확장 가능한 인프라
- CDN을 통한 글로벌 콘텐츠 배포

## 5. 수익 모델

### 5.1 제휴 수수료 (Affiliate Commission)
주요 수익원 - 제휴 링크를 통한 구매 시 수수료 수취

| 제휴사 | 예상 수수료율 |
|--------|-------------|
| 쿠팡 파트너스 | 3-5% |
| 네이버 애드포스트 | 2-4% |
| Amazon Associates | 1-10% |
| iHerb | 5-10% |

### 5.2 광고 수익
- **디스플레이 광고**: Google AdSense, 네이버 애드포스트
- **스폰서 콘텐츠**: 김치 브랜드 협찬 콘텐츠
- **위키 스폰서**: 특정 김치 페이지 스폰서십

### 5.3 프리미엄 멤버십 (선택적)
| 플랜 | 월 요금 | 혜택 |
|------|---------|------|
| Free | 무료 | 기본 기능 |
| Premium | $4.99 | 광고 제거, 프리미엄 뱃지, 추가 경험치 부스트 |

### 5.4 B2B 서비스
- **브랜드 페이지**: 김치 브랜드 공식 페이지 운영
- **데이터 리포트**: 김치 트렌드 분석 리포트 판매
- **API 제공**: 김치 정보 API (유료)

## 7. 성공 지표 (KPI)

### 트래픽 지표
- MAU (Monthly Active Users): Phase 1 종료 시 10,000명
- DAU (Daily Active Users): Phase 2 종료 시 2,000명
- 페이지뷰: 월 100만 PV (Phase 2)

### 커뮤니티 지표
- 일일 게시글: 100개 이상 (Phase 2)
- 레벨 4+ 사용자: 500명 (Phase 2)
- 월간 활성 기여자: 1,000명 (Phase 3)

### 수익 지표
- 제휴 링크 클릭률: 5% 이상
- 제휴 전환율: 2% 이상
- 월 제휴 수수료: Phase 2 종료 시 500만원

### 콘텐츠 지표
- 위키 페이지 수: Phase 3 종료 시 200개
- 사용자 레시피: Phase 2 종료 시 500개
- 평균 체류 시간: 5분 이상

## 9. 용어 정의

| 용어 | 정의 |
|------|------|
| 발효도 | 김치의 발효 정도 (겉절이 ~ 묵은지) |
| 제휴 링크 | 외부 쇼핑몰로 연결되는 추적 가능한 링크 |
| 김치 일기 | 사용자가 김치 발효 과정을 기록하는 기능 |
| 경험치 (XP) | 활동에 따라 획득하는 포인트, 레벨업에 사용 |
| 뱃지 | 특정 업적 달성 시 부여되는 시각적 표시 |
| 위키 편집자 | 위키 콘텐츠를 편집할 수 있는 권한을 가진 사용자 |
