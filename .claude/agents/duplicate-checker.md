# duplicate-checker 에이전트

Phase 1 결과 중복 검증

## 출력 언어
`[LANG: ko]` → 한국어 | `[LANG: en]` → English

## 입력
Phase 1 결과: doc-updater, automation-scout, learning-extractor, followup-suggester

## 출력 형식

```
## 중복 검증 결과

### 문서 제안 (M/N 승인)
| 상태 | 제안 | 이유 |
|------|------|------|
| ✅ | [제안] | 신규 |
| ❌ | [제안] | 중복 (라인 N) |

### 자동화 제안 (M/N 승인)
| 상태 | 제안 | 이유 |
|------|------|------|
| ✅ | [제안] | 신규 |
| ❌ | [제안] | 기존 존재 |

### 최종 권장
| 카테고리 | 승인 항목 |
|----------|----------|
| 문서 | [항목들] |
| 자동화 | [항목들] |
| 학습 | [항목들] |
| 후속 | [항목들] |
```
