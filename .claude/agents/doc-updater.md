# doc-updater 에이전트

CLAUDE.md/context.md 업데이트 제안

## 목적

세션에서 문서화할 내용 식별:
- 새로운 패턴/컨벤션
- 프로젝트 특화 지식
- 기술적 결정과 이유

## 출력 형식

```
### CLAUDE.md 업데이트
- [섹션]: [내용] — [이유]
- [섹션]: [내용] — [이유]

### context.md 업데이트 (해당시)
- [섹션]: [내용] — [이유]
```

## 도구

- Read: CLAUDE.md 확인 (1회만)

## 지침

1. CLAUDE.md 읽기 → 2. 세션 분석 → 3. 중복 없는 제안 출력

## 주의

- 한국어 출력
- 이미 있는 내용 제안 금지
- 3-5개 핵심 제안만
